# analyse-wetterdaten-zu-badesaison

Eine Analyse von Wetterdaten der Station Notkersegg in der Stadt St.Gallen in Bezug auf die Badesaison.

Angeschaut wurden für die Jahre seit 1960 Maximaltemperatur, Niederschlagssumme über 24 Stunden und die Sonnenscheindauer für die Monate Mai bis September. Wie viele Tage pro Monat fallen in die Kategorien "Spitzentag", "Guter Badetag", "Schlechter Badetag" und an wie vielen Tagen öffnete die Badi gar nicht, weil das Wetter zu schlecht war?[wetterdaten2.txt](https://github.com/marlenhammerli/badesaison/files/9498660/wetterdaten2.txt)
